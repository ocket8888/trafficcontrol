servers/totals¶

GET¶

Retrieves a count of each type of server across all CDNs.

Auth. Required

Yes

Roles Required

None

Response Type

Array

Request Structure¶

No parameters available.

Response Structure¶

count

The number of servers of this type configured in this instance of Traffic Ops

type

The name of the type servers herein counted
